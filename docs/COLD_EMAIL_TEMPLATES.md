## Outreach Strategy

**Where to send:**

1. **LinkedIn**: Agency owners, SEO managers, freelancers
2. **Twitter**: SaaS founders, indie hackers, developers
3. **Reddit**: r/SEO, r/freelance, r/webdev (as comments on relevant threads)
4. **Email**: Cold outreach to agencies (Template 1)
5. **ProductHunt**: Launch with "Show HN" on HN if applicable

**Timing:**

- Send Template 1 (agencies) on Tuesday-Thursday, 9am-11am their timezone
- Send Template 2 (freelancers) on Tuesday-Thursday, 5pm-7pm
- Twitter: Post daily at 9am UTC
- Blog: Publish on Tuesday, cross-post to LinkedIn Friday

**Follow-up:**

- If no reply after 5 days: Send gentle follow-up
- If reply: Schedule 15-min demo call
- If "not interested": Add to nurture list (email monthly updates)

## Key Messaging Points

When responding to objections:

**"But I use Semrush for [feature]"**
→ "We support that too. API docs: [link]. Want to test it first?"

**"Semrush has better data"**
→ "We use the same data sources. The difference is we don't charge extra for API access."

**"My team is already trained on Semrush"**
→ "Dashboard works the same way. API takes 2 hours to integrate if you want it."

**"What about support?"**
→ "Direct email from me. No support ticket queues. You get me."
