## 3. Launch Day Strategy (12-hour blitz)

**6am UTC Launch**
- Post product
- Include launch message (see below)
- Start responding to every comment within 5 minutes

**Throughout the day:**
- Every comment gets a reply (you + team)
- Every question gets a demo link
- Every concern gets addressed with data
- Offer special "ProductHunt exclusive" discount: 50% off first month

**24 hours after launch:**
- Message all commenters with "thanks for feedback" + link to demo
- If they mention specific features: "That's in the API docs: [link]"

**Pro tip:** ProductHunt ranking is based on hourly votes. You want steady momentum, not a spike then drop. Pace your team supporters throughout the day.
